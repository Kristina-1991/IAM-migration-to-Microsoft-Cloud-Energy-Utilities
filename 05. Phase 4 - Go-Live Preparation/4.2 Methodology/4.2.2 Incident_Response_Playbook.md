# Incident Response Playbook

**Version:** 1.0

**Owner:** IAM CoE

**Review:** Quarterly 

## 1. Incident Classification Matrix 

| Severity | Impact                                      | Examples                                   | Initial Response | Resolution Target |
| -------- | ------------------------------------------- | ------------------------------------------ | ---------------- | ----------------- |
| SEV 1    | >50 users, production down, security breach | Entra ID outage, mass provisioning failure | 5 min            | 4 hours           |
| SEV 2    | 10-50 users, degraded service               | SSO failure specific app, sync delays      | 15 min           | 8 hours           |
| SEV 3    | <10 users, single app impact                | Access request delay, single user lockout  | 1 hour           | 24 hours          |
| SEV 4    | Informational                               | Configuration change, minor performance    | 4 hours          | 3 days            |

## 2. Playbooks by Incident Type 

**SEV 1: Entra ID Outage**

1. T+0: Activate War Room (5 min)
2. T+5: AD fallback authentication (15 min)
3. T+15: Sentinel triage + Microsoft support (30 min)
4. T+60: Executive notification + comms plan
5. Resolution: Primary restoration OR sustained AD fallback

**SEV1: Mass Provisioning Failure**

1. T+0: Pause Workday sync (5 min)
2. T+5: Manual provisioning critical users (30 min)
3. T+30: Entra ID logs analysis via Sentinel
4. T+60: Sync fix + re-run failed jobs
5. T+120: Validation + user notification

**SEV 2: SSO Application Failure**

1. T+0: Verify Entra ID health (15 min)
2. T+15: AD direct auth bypass (30 min)
3. T+45: App-specific troubleshooting
4. Resolution: App config fix OR app rollback

## 5. Post-Incident Review (Mandatory) 

| Timeline | Deliverables                 |
| -------- | ---------------------------- |
| D+1      | Incident report + root cause |
| D+3      | Lessons learned workshop     |
| D+7      | Corrective actions assigned  |
| D+30     | Playbook update              |
